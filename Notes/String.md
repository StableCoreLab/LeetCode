# String

## 当前题目对应

### `5. Longest Palindromic Substring`

核心方法是中心扩展：

- 以单个字符为中心，处理奇数长度回文
- 以两个字符之间为中心，处理偶数长度回文

返回值通常需要记录：

- 当前最长长度
- 当前最长子串起点

### `30. Substring with Concatenation of All Words`

这是字符串题里很典型的“定长切片 + 哈希 + 滑动窗口”组合。

关键观察：

- 所有单词长度一致
- 所以可以把字符串按固定长度切块，而不是按字符逐个移动

## 常用操作

```cpp
string t = s.substr(pos, len);
int n = static_cast<int>(s.size());
```

## 易错点

- `substr(pos, len)` 的第二个参数是长度，不是结束下标。
- 混用 `size_t` 和 `int` 时要注意比较与减法结果。
- 回文题不能只检查奇数中心，偶数中心也必须覆盖。

## 与其他专题的关系

- 回文常和双指针结合
- 子串统计常和滑动窗口、哈希结合
- 变位词分组常和排序结合
