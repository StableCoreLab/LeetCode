# Two Pointers

## 适用场景

- 数组有序后，左右收缩寻找目标
- 需要在 `O(n)` 或 `O(n^2)` 内完成组合搜索
- 回文判断或中心扩展

## 当前题目对应

### `15. Three Sum`

做法：

1. 先排序
2. 固定第一个数 `nums[i]`
3. 用两个指针 `left`、`right` 在右侧区间找 `-nums[i]`

判断逻辑：

- 和为 `0`，记录答案，随后左右都要去重
- 和小于 `0`，`left++`
- 和大于 `0`，`right--`

为什么排序后可以双指针：

- 当前和偏小，只能通过增大左指针让总和变大
- 当前和偏大，只能通过减小右指针让总和变小

### `5. Longest Palindromic Substring`

中心扩展本质上也是双指针：

- 以一个字符为中心，向两边扩展，处理奇数长度回文
- 以两个字符之间为中心，向两边扩展，处理偶数长度回文

## 模板

```cpp
int left = start, right = end;
while (left < right) {
    if (满足条件) {
        ++left;
        --right;
    } else if (需要更大) {
        ++left;
    } else {
        --right;
    }
}
```

## `Three Sum` 的去重规则

- 固定值去重：`nums[i] == nums[i - 1]` 时跳过
- 找到答案后：
  - `left` 跳过重复值
  - `right` 跳过重复值

这一步决定了答案里是否会出现重复三元组。

## 复杂度

### `Three Sum`

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(1)`，不计答案

### 中心扩展回文

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(1)`

## 易错点

- 排序后忘记做去重。
- `Three Sum` 中已经出现 `nums[i] > 0` 时可以提前结束。
- 回文中心扩展时，奇偶中心都要检查。
